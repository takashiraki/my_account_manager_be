# ユーザー編集

## Index

### 晴れの日

管理者は、一覧画面より『編集する』をクリックする。
システムはURLから`id`を取得する。
システムは`id`が36文字か否かを確かめる。
システムは対象のユーザーが存在するか否かを確認する。
システムはユーザー情報を取得する。
システムはユーザー情報を返す。

![](/documents/images/EditUserUseCase.png)

### 雨の日
| 事象                           | 処理    | 備考                                  |
| ------------------------------ | ------- | ------------------------------------- |
| `id`が36文字ではない場合       | 429返す |                                       |
| 対象のユーザーが存在しない場合 | 404返す | `result`で`request_error`を持たせる。 |
